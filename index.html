Теги Атрибуты Описание Пример
<p> - Параграф
<ul> - Маркированный список
<li> - Элемент списка
<ol> - Нумерованный список
- <title> Всплывающий блок
- align="center,right" Выравние текста по центру или справа
<b> - Жирное выделение текста
<strong> - Логическое,жирное выделение текста
<i> - Курсивное выделение текста
<em> - Логическое,курсивное выделение текста
<h1-h6> - Заголовки
<table> - Граница таблицы
<tr> - Создание строки таблицы
<thead> - Шапка таблицы
<td> - Создание ячейки
<th> - Создание ячейки с заголовком
<tbody - Тело странцы
<tfoot> - Подвал таблицы
- <colspan> Обьединие ячейки по горизонтали
- <rowspan> Обьедние ячеек по вертикали
<a> href="" Указание страницы на переход
<a href="#"> <id="#"> Ссылка-якорь
<pre> <b,i,em,strong> Сохрание текста со всеми пробелами
<code> - Вставка кода в текст
<span> - Вставка текста(не розбрался полностью)
<img> src="" alt="" Вставка картинки <img src="https://example.com/images.png" alt="Аналитика компании за 2007 год">
- src="" Указывает путь картинки
